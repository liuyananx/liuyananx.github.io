# 关于我

你好，我是 **lynx**。

这里不是一份固定简历，也不是某个职业身份的说明书。

我会把阶段性的工作、生活、兴趣、想法和变化放在这里。以后它可能会长成很多种样子，我也不想太早给它定死。

## 关键词

::: info
变化 / 记录 / 工作 / 生活 / 兴趣 / 尝试 / 个人空间
:::

## 现在可能会出现的内容

- 工作里的项目和经验
- 生活里突然想记录的东西
- 新工具、新想法、新兴趣
- 一些不成体系但当下真实的想法
- 以后可能完全不一样的方向

## 我想把这里做成什么

它不是一次性写完的网站，而是一个可以慢慢生长的个人空间。

后续你可以把它当成：

- 阶段性的自我记录
- 一个公开但不必过分正式的主页
- 一些想法和作品的收纳处
- 一个允许变化发生的地方

## 后续可以补充

- 一张个人头像或喜欢的图片
- 某个阶段的自我介绍
- 喜欢的图片、音乐、书、地方
- 做过的项目或体验过的工作
- 当下正在想的事
- 不想被条条框框限制的部分
