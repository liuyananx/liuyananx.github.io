# 随想

这里放零散的想法、临时的念头、不成体系的记录。

不用写得很完整，也不用一定有结论。

## 怎么写

打开 GitHub.dev：

[https://github.dev/liuyananx/liuyananx.github.io](https://github.dev/liuyananx/liuyananx.github.io)

在下面这个目录中新建 `.md` 文件：

```text
docs/thoughts/
```

比如：

```text
docs/thoughts/2026-07-06-random.md
```

内容可以这样写：

```md
# 今天的一点想法

2026-07-06

今天突然想到……
```

提交后，网站会自动更新。
