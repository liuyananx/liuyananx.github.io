# Things

这里放各种“东西”。

它可以是项目，也可以是兴趣；可以是工具，也可以是一段经历。先不急着分类，先让内容有地方待着。

## 工作相关

- [项目记录](/projects)：做过或正在做的项目、页面、系统和作品。
- 技术笔记：以后可以放到 [Notes](/notes/) 里慢慢整理。

## 生活相关

这里可以放：

- 喜欢的地方
- 想去的城市
- 某段时间的照片
- 最近喜欢的音乐、电影、书
- 一些看起来没什么用但自己想保存的东西

## 尝试和实验

这里可以放：

- AI 工具体验
- 新技术尝试
- 小工具、小页面、小想法
- 以后突然想做的任何东西

::: tip
如果一个东西暂时不知道该放哪，就先放这里。
:::
