# 笔记目录

这里用来放相对完整一点的笔记、文章和长期文档。

如果只是临时想法、碎片记录、几句话的念头，可以先放到 <a href="/thoughts">随想</a>。

## 推荐分类

- 技术笔记
- 项目复盘
- 读书记录
- 工作流程
- 生活随笔

你可以直接在当前目录下新增 `.md` 文件。

## 在线写作

可以打开 GitHub.dev 在线编辑：

[https://github.dev/liuyananx/liuyananx.github.io](https://github.dev/liuyananx/liuyananx.github.io)

然后在这个目录下新建 Markdown 文件：

```text
docs/notes/你的文章名.md
```

也可以使用站内写作页：

[打开写作后台](/admin)
