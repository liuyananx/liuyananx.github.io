# Now

这里记录我最近正在做、正在想、正在体验的事情。

它不需要长期正确，只代表这个阶段。

## 最近

- 搭建这个个人网站，让它先成为一个可以持续变化的容器。
- 尝试把工作、生活、兴趣和想法放在同一个空间里。
- 不急着把自己定义成某一种固定身份。

## 正在想

我好像不太适合被太多条条框框限制。

如果一个网站太像简历，它会把人压缩成一个职业标签；如果太像博客，它又会要求每篇内容都成体系。

所以这里先保持开放：想到什么、做了什么、经历了什么，都可以慢慢放进来。

## 可以随时替换

这个页面以后可以变成：

- 最近在做的项目
- 最近想去的地方
- 最近喜欢的东西
- 最近正在学的东西
- 最近不想解释但想留下的状态
